# SRAMS Enterprise Transformation Plan

> **Goal:** Transform the existing working SRAMS into a CAMIS-inspired enterprise education platform
> **Stack:** React 19 + Vite + Tailwind CSS + Express 5 + PostgreSQL (PREN)
> **Date:** 2026-07-12
> **Decisions:** All 13 phases. Shared portals with RBAC (Admin portal serves Head Teacher, DoS, Deputy HT, etc. with permission-based filtering). Parent gets a 4th portal.

---

## Current State Assessment

The existing SRAMS has a **working foundation**:
- **Backend:** 19 controllers, 19 route files, all mounted. PostgreSQL schema with 16+ tables. Auth with JWT/HttpOnly cookies. Student PIN login functional. Dashboard stats endpoints for all 3 roles.
- **Frontend:** 33 page files across 3 portals (Admin 18, Teacher 6, Student 7). Login with portal toggle. Framer Motion animations. CascadeSelector. ReportCardPDF component. All pages are functional CRUD.
- **Missing:** Refresh tokens, forgot/reset password, email verification, OTP, 2FA, audit logs, role-based resource filtering (15 roles), timetables, marks workflow (multi-approval), PDF/Excel report card generation, custom template builder, competencies, parent portal, and many student management features.

```javascript
export const requirePermission = (...permissions) => {
  return async (req, res, next) => {
    const rolePermissions = ROLE_PERMISSIONS[req.userRole];
    if (!rolePermissions || !permissions.some(p => rolePermissions.includes(p))) {
      return res.status(403).json({ error: 'Insufficient permissions' });
    }
    next();
  };
};
```

// Axios interceptor for auto-refresh
api.interceptors.response.use(
  (response) => response,
  async (error) => {
    if (error.response?.status === 401 && !error.config._retry) {
      error.config._retry = true;
      await api.post('/auth/refresh');
      return api(error.config);
    }
    return Promise.reject(error);
  }
);
